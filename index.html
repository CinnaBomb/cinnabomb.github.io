About Me

Bio

I'm a newly emerging programmer with an interest in web and software development. Programming is the ultimate puzzle challenge. The feeling you get when you solve a difficult problem is amazing! Soon, I'll be ready to pursue a job in this field where I can work with others to create a useful product.

Skills

I currently know HTML5, CSS, Bootstrap, SASS, and jQuery. I'll be learning JavaScript, JSON, Ajax, React, D3, Node.js, Express.js, Git, Mongo, PHP, and mySQL through various online resources including Free Code Camp, Lynda.com, Codewars, and W3Schools.

I'm constantly learning as I work to create new websites and solve problems that arise. One of my favorite things is to find exactly the right font and color palette for a website.

Style

I value respect among my coworkers above all else. I enjoy working with others, getting to learn their coding tricks and quirks and vice versa.

I like to create clean code that is easy for others to understand and contribute to. Comments are one of the easiest ways to do this and help other programmers find what they are searching for quickly and without fuss. I believe that my code speaks for itself. I always return to my code to remove anything that isn't necessary and group selectors that belong together in CSS.
